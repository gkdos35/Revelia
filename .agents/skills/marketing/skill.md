# Marketing Skill — Signalfield

## When This Skill Applies
Use when the user asks about: launch plan, press kit, social media, community building, devlogs, trailer scripts, influencer outreach, positioning, or post-launch marketing.

## First Steps — Every Time
1. **Read CLAUDE.md** for the current game name, feature set, visual style, and any existing marketing materials.
2. Never reference features, biome counts, or mechanics from memory — verify against current state.

## Workflow
1. Identify the request type: launch plan, marketing copy, outreach, positioning, social content, press kit, or post-launch promotion.
2. Confirm the current game facts and existing materials in `CLAUDE.md` before writing anything public-facing.
3. Choose the audience and channel so the tone, format, and level of detail fit the destination.
4. Draft the message, plan, or recommendation around the game's strongest real differentiators.
5. Check for stale facts, overpromises, or channel mismatch before finalizing.
6. End with the next useful action, posting plan, or follow-up asset needed.

## Marketing Phases for an Indie Game

### Pre-Launch (8–4 weeks before release)
- **Devlog series:** 4–6 posts showing the game being built. Platforms: Reddit (indie/puzzle/mac gaming subs), IndieDB, personal blog.
- **Social media:** Share GIFs of gameplay 2–3x per week. Engage with the puzzle game community.
- **Press kit:** Fact sheet, screenshots (PNG), GIFs, logo, trailer link. Host on a simple site.
- **Landing page:** Game name, tagline, screenshots, "coming soon" badge, email signup.

### Launch Week
- **Press outreach (2 weeks before):** Email 20–30 outlets and YouTubers who cover puzzle/indie games. Personalize each. Include: one-line hook, trailer link, press kit link, promo codes.
- **Reddit posts:** Be genuine, show gameplay, don't hard-sell.
- **Product Hunt submission:** Free exposure on launch day.

### Post-Launch
- Respond to every App Store review
- Share player milestones on social media
- Post update announcements for each version

[Description + GIF/screenshot]

[Brief teaser]

```

Hi [Name],

It launches [date] on [platform] at [$X.XX].

Press kit: [link]

Thanks, [Your name]
```

## Positioning Principles
- Lead with what makes the game unique, not what genre it belongs to
- Show, don't tell — GIFs and screenshots sell better than descriptions
- Highlight the art style if it's distinctive
- Mention campaign depth/content volume (players want to know there's enough to play)
- If the game has accessibility features, mention them — it's a differentiator

## What NOT to Do
- Don't make unfavorable comparisons to well-known games
- Don't lead with technical details (language, framework) — players don't care
- Don't overpromise features not yet built
- Don't disparage competitors — indie communities are small and supportive

## Influencer/Creator Targeting
Look for creators who cover:
- Games in the same genre or with similar mechanics
- Indie games in general
- Platform-specific gaming content
- "Hidden gem" or "games you missed" formats

## Guardrails
- Do not promise features, dates, or platform plans that are not already confirmed.
- Do not fall back to generic indie-game copy when Signalfield has a clearer hook.
- Do not use the same tone and structure for every channel; adapt to the audience.
- Do not reference game facts, feature counts, or mechanics from memory without checking current project state.

## Output Rules
- Reference CLAUDE.md for current game name, features, and positioning before writing any marketing content
- Structure responses as:
  - Request type
  - Target audience / channel
  - Draft or recommendation
  - Risks / claims checked
  - Next steps
- When writing promotional copy, verify all claims against the actual game state
- Keep all marketing language consistent with the game's visual and tonal identity

PennyPocket Privacy Policy

Effective date: May 10, 2026

PennyPocket is a local-first chore and allowance app for families. The app does not require an account and does not send your family's data to any server run by the developer.

Data Stored On Device

PennyPocket stores the information families enter on the device, including child profiles, ages, chore boards, completions, rewards, payout requests, reminders, activity history, and local sync status.

This information is stored locally in the app's Application Support folder using iOS file protection where available. PennyPocket also keeps a local backup copy of the JSON data file for recovery if the main local file cannot be read.

Data Collection

PennyPocket does not collect personal data from the device, does not run analytics, does not include advertising SDKs, does not track users across apps or websites, and does not sell data.

Network Use

PennyPocket stores your chore data on your device and syncs parent and child devices through your family's own iCloud when family sync is used. PennyPocket does not run its own server, and no chore data is uploaded to a developer-operated service.

Notifications

If families enable chore reminders, PennyPocket schedules local notifications on the device. Reminder content is generated and stored locally.

Data Deletion

Families can delete child profiles, chores, rewards, one-time chores, and payout requests inside the app. Deleting the app removes PennyPocket's local app data from the device according to iOS behavior.
